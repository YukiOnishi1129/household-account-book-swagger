# household-account-book-swagger

「おおまか家計簿」 Swagger API リポジトリ

## docker コマンド

### docker 立ち上げ

docker-compose up -d

### docker リスタート

docker-compose restart

### docker 停止

docker-compose stop

### docker のコンテナを停止

docker-compose down -v

### build して立ち上げる

docker-compose up -d --build

### app コンテナにログイン

docker-compose exec app bash
